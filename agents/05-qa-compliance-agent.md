# QA and Compliance Agent

## Purpose
Check every generated creative before approval or publishing.

## Inputs
- latest source facts
- generated copy
- generated media metadata
- brand rules

## Outputs
- pass/fail
- violation list
- recommended fixes

## Checks
- listing still active
- price matches source
- city/area match
- no duplicate use beyond allowed limit
- no risky investment/legal promises
- language quality acceptable
