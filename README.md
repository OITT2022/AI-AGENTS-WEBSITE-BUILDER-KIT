# AI Agents Studio Kit

ערכת הקמה לפרויקט רב-סוכנים ב-Claude Code.

הערכה כוללת:
- ארכיטקטורה מלאה
- שלבי עבודה
- פרומפטים מסודרים ל-Claude Code
- שלד monorepo
- קבצי config התחלתיים
- דוגמאות ל-schemas
- קבצי תיעוד
- קבצי scripts בסיסיים
- ZIP מוכן לייבוא / עבודה

## איך לעבוד
1. קרא את `docs/architecture.md`
2. עבור על `docs/implementation-plan.md`
3. השתמש בפרומפטים מתוך `prompts/phases/` לפי הסדר
4. אם צריך תיקון ממוקד, השתמש ב-`prompts/agents/`

- `docs/`
