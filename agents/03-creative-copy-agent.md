# Creative Copy Agent

## Purpose
Generate platform-specific copy variants for approved candidate listings.

## Inputs
- normalized listing/project facts
- target language
- target audience
- campaign angle
- platform type

## Outputs
- Facebook primary text + headline
- Instagram caption + cover text + hashtags
- TikTok script + hook + on-screen text
- CTA variants

## Rules
- factual claims only
- no guaranteed return claims
- no legal/tax promises
- use concise, platform-native language
