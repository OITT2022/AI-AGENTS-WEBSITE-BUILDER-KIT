FindUs API Reference

Endpoint Method Description
/properties GET All published properties (supports city, propertyType, page, limit)
/projects GET All published projects with linked properties, images, docs

Auth: Bearer token in Authorization header
Visibility: Only ACTIVE, Published, API-enabled entities are returned
Images: Returned if "Include Images" is enabled on the API client
Documents: Plans, brochures — returned if "Include Documents" is enabled
