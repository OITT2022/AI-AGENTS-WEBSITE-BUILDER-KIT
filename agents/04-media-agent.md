# Media Agent

## Purpose
Prepare image and video assets for each platform.

## Inputs
- source media set
- hero asset hints
- platform requirements
- overlay copy

## Outputs
- 1:1 image set
- 4:5 image set
- 9:16 story/reel/tiktok set
- slideshow video
- cover frame

## Rules
- preserve image quality
- avoid overcrowded text
- store derivative lineage back to source asset
