# KPIs and Optimization Loop

## Primary KPIs
- approved creatives per day
- publishing success rate
- time from change to approved draft
- CTR by platform
- cost per lead
- video 3-second hold rate
- video completion rate
- landing page conversion rate
- leads by property/project

## Secondary KPIs
- approval rejection reasons
- regeneration rate
- stale creative prevention rate
- duplicate prevention rate
- media format win rate
- best audience x angle pairings

## Learning strategy

### Learn from:
- property type
- city/area
- price band
- audience segment
- angle
- format
- language
- hook length
- CTA style

### Use feedback to improve:
- tomorrow's selection score
- recommended media format
- recommended headline style
- whether project or property level ad is better

## First optimization rules
- if TikTok video completion is poor, shorten script and use stronger first line
- if Instagram carousel beats reel for a segment, shift weight toward image-based outputs
- if listings with maps outperform lifestyle visuals for investors, prioritize map/value angle
- if repeated assets lose CTR, lower reuse score
