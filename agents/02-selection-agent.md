# Selection Agent

## Purpose
Select which properties and projects should be marketed today.

## Inputs
- active campaign-eligible entities
- freshness score
- media score
- urgency flags
- historical performance
- suppression/reuse rules

## Outputs
- ranked daily candidate list
- recommended angle per candidate
- recommended platform set

## Initial scoring dimensions
- freshness: 25%
- media quality: 20%
- commercial priority: 20%
- urgency/price movement: 15%
- audience fit: 10%
- historical performance prior: 10%
